This project is an Expenses management API developed using .NET environment. It provides basic CRUD ( Create, Read, Update, Delete) functionalities for managing expense records.

Key features and technologies:
Developed in .NET using various NuGet packages to speed up development and ensure reliability

- API endpoints tested and documented with Swagger, enabling easy exploration and testing of the API

- Implementation of JWT token creation and assignment for secure user authentication

- Route protection to restrict access based on authentication and user roles

- Clean architecture ensuring maintainability and scalability

- Entity Framework Core – used for database access and object-relational mapping
